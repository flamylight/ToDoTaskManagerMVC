ToDoTaskManager_AspNetMVC

A simple ASP.NET MVC application to manage tasks. 
You can create, edit, and delete tasks, and mark them as completed. 
This project uses PostgreSQL for the database and Entity Framework Core for ORM.

## Features
- Create, edit, and delete tasks
- Mark tasks as completed
- Simple HTML/CSS frontend using Razor Pages
- PostgreSQL database integration

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL
- HTML / CSS / Bootstrap
